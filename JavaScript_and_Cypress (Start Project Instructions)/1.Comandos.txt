comandos devem ser efetuados no terminal do visual studio code

npm init --yes (iniciando o projeto, --yes significa yes automaticamente para todas as perguntas caso queira responder uma a uma
separadamente execute somente o npm init sem o --yes), após executar será criado um arquivo package.json

cls (Limpa a tela do terminal)

npm instal -D cypress (Instala o cypress na ultima versão disponivel)
npm instal -D cypress@4.1 (Instala o cypress em uma versão especifica que está setada no comando apos o @)

./node_modules/.bin/cypress open (Criar a Estrutura de pastas e abrir o cypress já com a estrutura de pastas)
Obs. Você pode executar mesmo que seja só para abrir pois ao executar o comando novamente ele vai verificar se ja existe o arquivo
de configurações global e se já existe a pasta cypress, se já existir ele só ira abrir, para não sobreescrever testes já escritos.
